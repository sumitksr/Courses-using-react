## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/sumitksr/courses-website.git
```

2. Navigate to the project directory
```bash
cd courses-website
```

3. Install dependencies
```bash
npm install
```

4. Start the development server
```bash
npm start
```
